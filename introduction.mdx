title: Введение
description: 'О проекте Caldera'
---

Caldera - это платформа для развертывания роллапов, которая позволяет запускать высокопроизводительные, настраиваемые, специфичные для каждого приложения роллапы с использованием фреймворков Arbitrum Nitro и Optimism Bedrock.

Каждый роллап Caldera поставляется с производительными узлами RPC, блокчейн-проводником, индексатором данных и мостовым интерфейсом к расчетному чейну. Роллапы Caldera также получают преимущества от <a href="https://caldera.xyz/integrations/">интеграции</a> с 40+ ведущими поставщиками инфраструктуры Web3.
